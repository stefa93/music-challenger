## Typography

The primary font is "Patrick Hand" for a distinct, handwritten feel. The default system font is used sparingly for long text.

<Typeset
  fontFamily='"Patrick Hand", cursive'
  fontSizes={['14px', '16px', '18px', '20px', '24px', '30px', '36px', '48px']}
  fontWeight="regular"
  sampleText="Songer makes music fun!"
/>

*   **Primary Font:** "Patrick Hand" (`font-handwritten`) - Used for headings, buttons, tags, titles, prices, list items.
*   **Body Font:** `system-ui` - Reserved for rare cases needing maximum readability.
*   **Emphasis:** `font-bold` is used for key info like prices and titles.
*   **Rotation:** Subtle rotation (`rotate-[-1deg]`, `rotate-[1deg]`, etc.) is applied to enhance the hand-drawn style.

## Spacing

Consistent spacing is maintained using Tailwind's standard scale. Common utilities include:
*   `gap-8`
*   `mb-6`
*   `space-y-3`
*   `p-4`, `px-3`, `py-2`

Refer to the [Tailwind Spacing Scale](https://tailwindcss.com/docs/customizing-spacing#default-spacing-scale) for details.

## Borders & Shadows

A distinct border and offset shadow style is used for interactive elements.

*   **Border:** `border-2 border-zinc-900 dark:border-white`
*   **Shadow:** Custom offset shadow `shadow-[4px_4px_0px_0px] shadow-zinc-900 dark:shadow-white` or theme extensions like `shadow-creative-button`.
*   **Hover/Active:** Shadows and positions often change on interaction (see `CreativeButton` or `CreativeCard` stories).
